初识TensorFlow

初期准备：

  • 安装好TensorFlow
  • 知道如何在Python中编程
  • 懂一点数组知识

TensorFLow提供多种APIs，从低级到高级，满足不同使用需求，越高级越容易学习和使用。下面的一些模型都可以用tf.contrib.learn高级API实现。

tf.train API
